# EasyTrips 🌍

EasyTrips is a full-stack travel and tour planning web application that allows users to explore destinations, manage profiles, and track travel insights in a clean and responsive interface.

---

## 🚀 Live Demo
👉 https://easytrips-n5o76ihsb-fs-projects-c4c8a5c5.vercel.app

---

## 🛠️ Tech Stack

### Frontend
- React (Vite)
- Tailwind CSS (v3)
- shadcn/ui
- React Router
- React Toastify

### Backend
- Node.js
- Express.js
- MongoDB
- JWT Authentication

### Deployment
- Vercel (Frontend)
- Backend hosted separately

---

## ✨ Features

- User authentication (JWT based)
- Profile management
- Update personal details (name, phone, address, GST)
- Travel insights dashboard
- Responsive UI with Tailwind CSS
- Secure API integration
- Clean and modular code structure
